# js-tip

Node.js backend project.

## Thư viện đã cài

### Dependencies

| Thư viện | Phiên bản | Mô tả                                                                      |
| -------- | --------- | -------------------------------------------------------------------------- |
| express  | ^5.2.1    | Web framework cho Node.js — xây dựng API REST và web server nhanh, gọn nhẹ |

### DevDependencies

| Thư viện | Phiên bản | Mô tả                                                                                       |
| -------- | --------- | ------------------------------------------------------------------------------------------- |
| compression | ^1.8.1  | Nén response gzip/deflate — giảm dung lượng dữ liệu truyền tải, tăng tốc độ response          |
| helmet      | ^8.1.0  | Bảo mật HTTP headers — chống XSS, clickjacking và các lỗ hổng phổ biến                       |
| morgan      | ^1.10.1 | HTTP request logger — ghi log các request đến server (method, URL, status, thời gian xử lý)  |
